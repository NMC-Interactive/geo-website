# Visual & UX Localization for China

> **GEO Preface:** This article covers visual design and UX localization for the Chinese market — essential for creating experiences that feel native to Chinese users. This detail article is part of our [Content Localization for China](/world-to-china/content-localization-china/) spoke within the [World-to-China GEO](/world-to-china-geo/) pillar. For Simplified Chinese, read [Simplified Chinese](/world-to-china/content-localization-china/simplified-chinese/). For local context, see [Adding Local Context](/world-to-china/content-localization-china/local-context/).

## Visual Localization

**Imagery:**
- Use photos of Chinese people in Chinese settings
- Avoid Western-only imagery
- Include diverse Chinese demographics
- Use culturally relevant scenes

**Color Considerations:**
- Red = luck, prosperity (positive)
- Gold = wealth, success (positive)
- White = mourning in traditional context (use carefully)
- Yellow = royalty, prestige (positive)

**Iconography:**
- Use QR codes prominently (WeChat Pay, following)
- Include WeChat/Alipay icons for payment
- Use familiar UI patterns
- Adapt icons for Chinese cultural understanding

## UX Localization

**Information Density:**
- Chinese users often prefer more information density
- Don't oversimplify — provide detail
- Use expandable sections for advanced info

**Navigation Patterns:**
- Mobile-first is essential
- Bottom navigation tabs work well
- WeChat Mini Program integration
- QR code scanning as primary entry point

**Payment Integration:**
- WeChat Pay and Alipay are essential
- QR code payments dominate
- UnionPay for card payments
- Avoid PayPal/credit card-only approaches
