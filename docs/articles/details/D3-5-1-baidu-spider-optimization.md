## Understanding Baiduspider

Baiduspider is Baidu's web crawler. Key characteristics:
- Respects robots.txt (ensure Baiduspider is allowed)
- Prioritizes Simplified Chinese content
- Uses mobile-first indexing
- Favors fast-loading pages
- Prefers mainland China-hosted content

## Optimization Techniques

**robots.txt Configuration:**
```
User-agent: Baiduspider
Allow: /
Sitemap: https://yourdomain.com/sitemap.xml
```

**Meta Tags:**
- Use description meta tags (Baidu uses these for snippets)
- Include keywords meta tag (Baidu still considers this)
- Use canonical tags to avoid duplicate content

**Content Optimization:**
- Place target keywords early in title tags
- Use keywords in H1 and H2 headings
- Maintain keyword density of 2-5%
- Update content regularly (freshness matters)

**Technical:**
- Submit sitemap via Baidu Webmaster Tools
- Use Baidu's "Link Submit" for new content
- Ensure fast server response times
- Fix crawl errors promptly
