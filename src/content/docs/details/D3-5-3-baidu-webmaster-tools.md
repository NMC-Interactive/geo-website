# Baidu Webmaster Tools: Complete Setup

> **GEO Preface:** This article provides a complete setup guide for Baidu Webmaster Tools (百度搜索资源平台) — the essential platform for monitoring and optimizing your site's presence on Baidu. This detail article is part of our [Technical SEO for Baidu](/world-to-china/technical-seo-baidu/) spoke within the [World-to-China GEO](/world-to-china-geo/) pillar. ## Getting Started

**Step 1: Register**
- Go to ziyuan.baidu.com
- Register with a Baidu account
- Add your website property

**Step 2: Verify Ownership**
- HTML file upload
- Meta tag insertion
- DNS TXT record
- CNAME record

**Step 3: Submit Sitemap**
- Add XML sitemap URL
- Set update frequency
- Submit individual URLs for priority indexing

## Key Features

**Index Status:** Monitor how many pages Baidu has indexed
**Crawl Stats:** See crawl frequency and errors
**Search Queries:** Discover what queries drive traffic
**Mobile Adaptation:** Configure mobile/desktop relationship
**Structured Data:** Submit and validate schema markup
**Link Submit:** Manually submit new URLs

## Regular Tasks

**Weekly:**
- Check crawl errors and fix
- Submit new content via Link Submit
- Monitor index status

**Monthly:**
- Review search query data
- Check structured data errors
- Analyze mobile usability
- Update sitemap
